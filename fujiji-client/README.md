# Fujiji Frontend

## Getting Started

Instal all dependencies:
```
npm install
```

Create new `.env` file and put api keys here:
```
cp .env.test .env
```

Running the application:
```
npm run dev
```

## Contributing

To run tests:
```
npm run test
```

To run eslint:
```
npm run lint
```

To build the project:
```
npm run build
```
